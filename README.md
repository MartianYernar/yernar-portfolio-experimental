# Yernar Mars — portfolio (experimental)

Experimental redesign exploring a different assembly of the same content:

- **Bento grid** Featured Work section (Apple-style mixed-size tiles) instead of rows
- **Spike/identity diagram** — visualizes Research + Robotics + Startups converging into one "Builder" identity
- **Live GitHub stats** tile, fetched client-side from the public GitHub API
- **Custom magnetic cursor**
- **Typing hero tagline** ("Researcher. Builder. Founder. Mentor.")

The main portfolio (github.com/MartianYernar/yernar-portfolio) is untouched; this
is a parallel experiment sharing the same images/CV via a copied `images/` and
`gallery.js`. No build step — plain HTML/CSS/JS.

## Run locally
Open `index.html` directly, or `python -m http.server` then visit `localhost:8000`.
